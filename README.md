# Voice Assistant Project

A simple Python voice assistant using speech recognition and text-to-speech.

## Features

- Listens for voice input
- Responds with spoken feedback

## Requirements

- Python 3.x
- `speech_recognition`
- `pyttsx3`
- `pyaudio`

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
